# Quotation
見積表作成、改善のための案を考える

# 状況
現在、すでに配布されている見積もり自動作成ツールが使いづらい、
そちらを編集できればよかったがロックがかかっている為こちらで変更できない、
現状はVBAで作成しているがほかの手段を用いてよいので新しい見積もり計算ツールを作成したい。DBはすでにある

# 考察
計算式などはすべて入っているため現状計算式やそれにかかわるものは触れないでほしい、計算箇所が変更されたなら許可後にいじるように
これまでは一つのシートの見積書（はみ出たらスクロール下のシートに転写）に入れていたが、
見積用のエクセルファイルを新たに作成。シートはページごとに分ける。シート名は部位名を起用(D12)
